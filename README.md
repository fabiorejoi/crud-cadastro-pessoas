# CRUD Cadastro de Pessoas

Projeto backend simples desenvolvido em Node.js com Express.
Criado com foco didático, para demonstrar a construção de uma API REST
e o uso do Git como ferramenta de versionamento ao longo do desenvolvimento.

## Tecnologias
- Node.js
- Express
- Git e GitHub

## Estrutura do projeto
A aplicação segue uma estrutura simples:

- O Express é responsável pelo servidor
- A API expõe rotas REST
- Os dados são armazenados em memória, apenas para fins de estudo

## Rotas disponíveis

### Criar usuário
`POST /usuarios`

### Listar usuários
`GET /usuarios`

## Testes da API
As rotas podem ser testadas utilizando ferramentas como:
- Thunder Client
- Insomnia

## Versionamento
O projeto utiliza Git para controle de versão.

Exemplos:
- `feat`: inclusão de funcionalidades
- `fix`: correções realizadas durante o desenvolvimento
- `docs`: documentação do projeto

## Uso educacional
Este repositório pode ser utilizado em sala de aula para demonstrar:
- Desenvolvimento incremental
- Importância de validar dados
- Leitura do histórico de commits
- Boas práticas iniciais de versionamento
